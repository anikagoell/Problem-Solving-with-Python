# Problem-Solving-with-Python

A collection of fundamental Python implementations covering core problem-solving concepts — palindromes, sorting algorithms, searching techniques, and more.

This repository is a live learning space. I'm actively writing and pushing code as I build my foundations, so expect it to grow consistently.

---

## What's inside

- Sorting algorithms — bubble, selection, insertion, merge, quick
- Searching — linear search, binary search
- String problems — palindrome, anagram, reversal, compression
- Array problems — two-sum, rotate, find duplicates, sliding window
- More being added regularly

---

Found a bug? Have a cleaner approach or a different logic for the same problem?
